# DevHub 🚀

Plataforma de portfólios para desenvolvedores. Crie seu perfil, adicione projetos e seja descoberto.

## Tecnologias

- React 18 + TypeScript
- React Router v6
- Styled Components
- Context API + localStorage
- Vite

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse: <http://localhost:5173>

## Deploy

```bash
npm run build
```

Deploy automático via Vercel ao fazer push para `main`.

## Funcionalidades

- ✅ Autenticação (registro, login, logout)
- ✅ Perfil editável (foto, bio, GitHub, LinkedIn)
- ✅ CRUD de projetos com tags de tecnologia
- ✅ Portfólio público compartilhável (`/portfolio/:userId`)
- ✅ Busca de desenvolvedores com filtros
- ✅ Sistema de favoritos
- ✅ Comentários nos portfólios
- ✅ Design responsivo (mobile, tablet, desktop)
